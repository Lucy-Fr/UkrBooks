UkrBooks — Voices of Ukraine in Contemporary Literature

This repository contains the source files for a multilingual website dedicated to contemporary Ukrainian literature.
The project presents authors, new books, and cultural developments in Ukraine after 2022.
The site is built using GitHub Pages and includes three language versions:

English

Français

Українська

The website aims to provide a clear and accessible gateway to the evolving cultural landscape of Ukraine, with an emphasis on literature created during and after the full-scale Russian invasion.

Structure

index.html — English homepage

fr.html — French homepage

ua.html — Ukrainian homepage

logo.png — website logo

Additional content pages will be added as the project evolves.

Purpose

To introduce readers to:

new works of Ukrainian contemporary literature

wartime and post-wartime narratives

Ukrainian authors and their voices

literary and cultural events in modern Ukraine

Technology

GitHub Pages (static hosting)

HTML + CSS

Fully multilingual navigation
